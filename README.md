# rad-xray
xray+rad批量主动扫描

## Usage: 

## 社区版用户

url一行一个放url.txt中，和rad放同文件夹

xray开启监听，

./xray webscan --listen 127.0.0.1:7777 --html-output report__datetime__.html

运行py

python3 rad+xray.py

## 高级版用户

url一行一个放url.txt中

browserscan.py和xray.exe和rad.exe放同一文件夹

运行py:

python3 browserscan.py
